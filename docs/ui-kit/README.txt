
BARSTOCK UI KIT (Claude Implementation Pack)

Includes:
- design-tokens.json → color/spacing/font tokens for UI
- brand.css → CSS variables ready to import
- barstock-logo.svg → placeholder logo vector

Usage:
1. Import brand.css globally.
2. Use tokens for consistency in Flutter, React, HTML, etc.
3. Replace logo with final production SVG when available.
4. Background photo should be licensed bar image.

Recommended stack:
Flutter mobile app + Web admin dashboard.
